GuitaraokeLeader for Android

Play guitar and sing with your friends

The lyrics and chords show in sync with the "Leader" on any "Follower" smartphone.


Installation, first run and songs

1. Only the Leader needs to install this android app. The Followers will use their existing internet browser.

2. This app is safe (no special permission is required on Android). But it is not on Google Play, so there are a few settings to allow the installation.
The android installation package is called APK and can be downloaded and installed from unknown sources.
Allow temporarily to install this APK: "Settings - Security - Install Unknown apps". Click on Chrome and "Allow from this source".

Download APK 3.4 for android

3. After download, install it. Because this package is not from Google Play, there will be several warnings:
Do you really want to download? Do you really want to install it? Do you really trust this developer?
It is good that Android helps us for our security. You just answer to these warnings to continue anyway.
The last question after installation is if you want to open the app. Answer "No", because first we need to prepare the local network.

4. The installation is finished and you can revert back "Settings - Security - Install Unknown apps". Click on Chrome and turn off "Allow from this source".

5. This app needs a local network to function. If you are connected to internet over wi-fi, you are already on a local network.
If you are connected to internet over "mobile data", you need to enable the mobile hotspot on your smartphone and share your internet connection.
This will create a local wi-fi network. Nobody will connect for now. We need it just to prepare the songs for the app and rehearse.

6. Open GuitaraokeLeader app for the first time. The app will immediately ask you to choose and allow access to a folder for the songs.
This is for security reasons. The app cannot see any other folder or files. Just the one you manually choose. Every time.
Because this is the first run, you have to first manually create a new Guitaraoke folder.
Go to the folder "Music" and create a new sub-folder "Guitaraoke".
Inside your Guitaraoke folder create a sub-folders called "Practice".
Finally click "Use this folder" and "Allow access" for the app. Very secure!

7. Now the app shows the main page. Click on "Download song", choose a song to download. The song will be downloaded in the background. Slow.
Later you can copy your mp4 songs with the android file manager to the "Music/Guitaraoke/Practice/" folder. That is much faster.
You can use any mp4 file, there are many guitaraoke and karaoke videos on the internet.

8. Click on the app title "GuitaraokeLeader" in the left upper corner to go back to the main page.

9. Click on the song and then "Play" and "Fullscreen". You can now practice your guitar and singing.

10. Click "back" to go out of FullScreen and then "Stop" to stop the song and go back to the main page.

11. You can now download more songs.

12. You already rehearse your guitar, now practice and rehearse with GuitaraokeLeader before going public !

13. Create a new folder under the Guitaraoke folder for example "Rock" or "Romantic" or "Traditional".
Use the android file manager to copy the songs you know well from the folder "Practice".
Create collections of songs for every event. You can choose the folder you want, everytime you start the app.


Go live - instructions for the Leader

1. Limit the internet traffic, change the "Mobile networks - Network mode" to "2G only".
Some smartphones when connected to wi-fi start immediately to download and sync enormous quantity of data.
But the 2G connection is so slow, that this will be barely visible on your data plan.

2. Enable the hotspot on your smartphone (share your internet connection, it will create a local wi-fi network).
Change the network name to "GuitaraokeLeader" and the password to "guitaraoke3.4".

3. Open GuitaraokeLeader app, choose your Guitaraoke folder like "Music/Guitaraoke/Rock" and allow access.

4. Click on QRCode. It will show 2 QRCodes. First is for the wifi connection, second to open the local web page.
All modern smartphone can use this 2 QRCodes. Let the followers read them with their smartphones.
When their application is running, they can share the same QRCodes to other people.
Older phones have to do it old-school: connect to your wifi with a password and
write manually the address in the browser 192.168....:8080 (it is in the upper right corner).

5. The Leader can see the number of Followers. Ask if everybody is connected!

6. The Leader clicks on a song, then clicks on the Play button and finally on the FullScreen button.

7. Start to play-a-long and sing-a-long everybody together.

8. You can use the back button to exit FullScreen.

9. You can Stop the song and return to the song list.

10.After using GuitaraokeLeader:
Change the network name and password to what you had before Guitaraoke. Then stop you mobile hotspot.
Revert the "Mobile networks - Network mode" to LTE.


Super short instructions for the Followers

1. The follower reads the first QRCode on the Leader phone for the wifi connection.
Older phones have to do it old-school, choose the network "GuitaraokeLeader" and enter the password "guitaraoke3.4".

2. The follower reads the second QRCode on the Leader phone. It will open the browser on the local web page.
Older phones must type manually the address in the browser from the right upper corner of the GuitaraokeLeader like "192.168.x.y:8080".

3. The followers can share these QRCodes with other followers, so the Leader is not overwhelmed.

5. The Follower clicks on the button "Full screen" and enjoys the play-a-long and sing-a-long.
